# MayDay Real Estate

A simple, static website for Hensdens Real Estate built with pure HTML5 and CSS3.

## Overview

This is a minimalist real estate website designed to establish online presence, showcase services, and provide contact information. The site follows a clean, professional design with no JavaScript or complex frameworks.

## Features

- ✅ **Single Page Design** - All content on one scrollable page
- ✅ **Mobile-First Responsive** - Optimized for all device sizes
- ✅ **Pure HTML/CSS** - No JavaScript or frameworks
- ✅ **Navy Blue Theme** - Professional color scheme (#0b2a4a)
- ✅ **Contact Form** - HTML5 validation with accessibility features
- ✅ **Services Section** - Clear listing of real estate services
- ✅ **Property Showcase** - Sample property listings
- ✅ **SEO Optimized** - Semantic HTML and meta tags
- ✅ **Print Friendly** - Dedicated print styles

## Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Responsive styling and layout
- **System Fonts** - Arial with fallbacks
- **Static Hosting** - Ready for Vercel deployment

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Muhammad-Anique/222-26437.git
   cd 222-26437
   ```

2. Open `index.html` in your browser or serve with a simple HTTP server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   ```

3. View at `http://localhost:8000`

## Deployment
